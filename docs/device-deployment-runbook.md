# Device Deployment Runbook

## Purpose

Provide a repeatable process for preparing a Windows laptop for a new or existing employee.

## Required Information

- Employee name
- Department
- Manager
- Start date
- Approved hardware type
- Required applications
- Required security groups or shared drives
- Work location

## Deployment Steps

1. Select available laptop from inventory.
2. Inspect physical condition.
3. Connect power and Ethernet.
4. Image or reset device using the approved Windows build.
5. Apply naming standard.
6. Install required updates.
7. Install Microsoft 365 apps and Teams.
8. Install department-specific applications.
9. Install printer drivers.
10. Confirm antivirus and disk encryption.
11. Test Wi-Fi, Ethernet, webcam, microphone, speaker, keyboard, trackpad, and ports.
12. Attach asset tag if missing.
13. Update asset inventory.
14. Add setup details to the Spiceworks ticket.
15. Stage laptop with charger, dock, headset, and monitor if required.

## Acceptance Criteria

- User can sign in.
- Outlook opens and mailbox is available.
- Teams opens and audio/video test succeeds.
- OneDrive sync starts.
- Required printer is available.
- Required applications launch.
- Device is encrypted and protected.
- Asset assignment is documented.
