# Spiceworks Ticket Standards

## Minimum Ticket Fields

- User name
- Contact method
- Location
- Device name or asset tag
- Category
- Priority
- Business impact
- Error message
- Troubleshooting steps
- Resolution or escalation note

## Good Ticket Summary Examples

- "Outlook repeatedly prompts for password after user password reset"
- "Conference Room A Logitech camera not detected in Teams"
- "New hire laptop setup for Finance employee starting May 11"
- "HP LaserJet Floor 1 print jobs stuck in queue"

## Poor Ticket Summary Examples

- "Computer broken"
- "Need help"
- "Printer issue"
- "Teams does not work"

## Resolution Note Standard

A good resolution note explains the cause, action taken, result, and any user communication.

Example:

> Cleared stuck print jobs, removed and re-added HP LaserJet M404, printed test page successfully, and confirmed user can print from Word and Outlook.
