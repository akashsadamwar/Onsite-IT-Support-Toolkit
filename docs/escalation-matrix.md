# Escalation Matrix

## Level 1 Owns

- Password reset guidance
- Basic Windows troubleshooting
- Printer queue and driver checks
- Teams audio/video setting checks
- Laptop setup and equipment recovery
- Ticket logging and user communication
- Basic network checks such as Wi-Fi, Ethernet, and IP address validation

## Escalate To Level 2

- AD group or permission changes requiring admin approval
- Microsoft 365 licensing issues
- Repeated Outlook authentication failures
- Device encryption failures
- Intune, SCCM, or imaging failures
- Printer server or print queue service issues
- Teams room device enrollment or firmware issues

## Escalate To Level 3

- Network outage or VLAN/routing issue
- Identity platform outage
- Security incident or suspicious login activity
- Server-side application failure
- Widespread Microsoft 365 service disruption
