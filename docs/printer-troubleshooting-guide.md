# Printer Troubleshooting Guide

## Common Symptoms

- User cannot see printer.
- Print job is stuck.
- Printer shows offline.
- User can print from one app but not another.
- Printer asks for authentication.
- Print quality is poor.

## Level 1 Checks

1. Confirm user, device name, printer name, and location.
2. Check whether other users can print.
3. Confirm network connectivity.
4. Confirm printer has paper and toner.
5. Clear stuck jobs from the local queue.
6. Restart the print spooler if appropriate.
7. Remove and re-add the printer.
8. Confirm default printer and paper size.
9. Print a test page.

## Escalate When

- Multiple users are affected.
- Printer server queue appears down.
- Driver package is missing or corrupt.
- Printer is unreachable by IP address.
- Hardware fault is shown on the device panel.
